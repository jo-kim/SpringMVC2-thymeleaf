# SpringMVC2-thymeleaf

+ Thymeleaf 사용 선언
  + ``` <html xmlns:th="http://www.thymeleaf.org"> ```
+ Thymeleaf 특징
  + 타임리프는 순수 HTML 파일을 웹 브라우저에서 열어도 내용을 확인할 수 있고
  + 서버를 통해 뷰 템플릿을 거치면 동적으로 변경된 결과를 확인할 수 있다. 
  + 서버 사이드 HTML 렌더링 (SSR) 
  + 스프링 통합 지원
  + 네츄럴 템플릿
     + 순수 HTML을 그대로 유지하면서 뷰 템플릿도 사용할 수 있는 타임리프의 특징
